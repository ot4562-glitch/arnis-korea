# Roadmap v0.9 to v1.2

## v0.9 Trace Editor MVP

- 한국어 GUI
- 프로젝트 저장/불러오기
- Naver Static Map API 테스트
- bbox와 스폰포인트 편집
- 레이어 직접 편집
- suggested layer 후보 생성과 사용자 승인
- accepted layer export
- synthetic preview export
- source policy와 validation report

## v1.0 Trace Editor Hardening

- 더 나은 선택/이동/수정 UX
- Static Map raster 타일 표시 개선
- 프로젝트 호환성 검사 강화
- 주소 검색 보조 기능 정리

## v1.1 Arnis Writer Connection

- accepted layer를 writer 입력으로 연결
- synthetic preview를 실제 writer 입력 포맷으로 승격
- GUI에 월드 생성 전 검수 단계 추가

## v1.2 Quality Loop

- 사용자 수정 이력
- layer diff와 review workflow
- 생성 월드 품질 리포트와 Trace Editor feedback loop
